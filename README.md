# 🛒 E-Commerce

상품 등록 및 결제 연동 커머스 웹 사이트 - Typescript, Tailwind CSS, React 프로젝트 (firebase 배포)
<br/><br/>

## 구현 기술

- NoSQL(firebase cloud store) 데이터(상품, 유저, 주문 내역) 관리 및 결제 연동
- 커스텀 훅, 전역 상태 관리 라이브러리 적용으로 비즈니스, 뷰 로직 분리
- 이미지 및 렌더링 최적화 적용
- 데이터 캐싱 및 등록 상품 실시간 데이터 동기화(react-query)

## 주요 기능

- 카테고리 검색 기능 (랜딩 페이지)
